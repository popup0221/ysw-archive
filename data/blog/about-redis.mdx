## Redis

- Redis(Remote Dictionary Server)는 오픈 소스 `인-메모리 데이터 구조 저장소`이다.
- `데이터베이스, 캐시, 메시지 브로커로 사용`한다.
- `key-value 구조를 기반`으로 다양한 데이터 구조(String, HashMap, List, Set 등)를 지원한다.
- `빠른 응답시간`을 제공
- 복제와 샤딩(Sharding)을 지원하여 확정성을 높일 수 있다.
  > 샤딩(Sharding)이란 `데이터를 조각내 분산 저장하는 데이터 처리 기법`으로 일관적 관리가 힘든 거대 데이터베이스나 네트워크를 작게 나눠서 저장하고 관리하는 기법이다.
- pub/sub 메시징 시스템을 통해 메시지 브로커로 사용있다.
- Lua 스크립트을 지원하여 서버 측 로직을 실행 할 수 있다.
- 프로그래밍 언어에서 사용할 수 있는 클라이언트 라이브러리를 제공한다.
- 세션 캐싱, 실시간 분석, 메시지 큐, 리더보드 등에 사용한다.
