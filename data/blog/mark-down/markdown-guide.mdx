# 마크다운이란?

`Markdown`은 웹에서 텍스트를 스타일링하는 방법입니다. 문서의 표시 형식을 제어하며, 단어를 굵게 하거나 기울임, 이미지를 추가하고 목록을 만드는 등 다양한 작업을 할 수 있습니다. 대부분의 경우 Markdown은 단순한 텍스트에 몇 가지 특수 문자를 추가하는 방식입니다.

## 헤더

```
# 이것은 h1 태그입니다

## 이것은 h2 태그입니다

#### 이것은 h4 태그입니다
```

# 이것은 h1 태그입니다

## 이것은 h2 태그입니다

#### 이것은 h4 태그입니다

<br />

## 강조

```
_이 텍스트는 기울임입니다_

**이 텍스트는 굵게 표시됩니다**

_기울임과 **굵게**를 같이 사용할 수 있습니다_
```

_이 텍스트는 기울임입니다_

**이 텍스트는 굵게 표시됩니다**

_기울임과 **굵게**를 같이 사용할 수 있습니다_

   1. 항목 3b

## 할 일 목록

```
- [x] 목록 구문 필요 (모든 순서 또는 비순서 목록 지원)
- [x] 완료된 항목
- [ ] 완료되지 않은 항목
```

- [x] 목록 구문 필요 (모든 순서 또는 비순서 목록 지원)
- [x] 완료된 항목
- [ ] 완료되지 않은 항목

<br />

## 표

단어 목록을 작성하고 하이픈 `-`으로 첫 줄을 나눈 후, 각 열을 파이프 `|`로 구분하여 테이블을 만들 수 있습니다:

```
| 첫 번째 헤더         | 두 번째 헤더         |
| ---------------------| ---------------------|
| 첫 번째 셀의 내용    | 두 번째 셀의 내용    |
| 첫 번째 열의 내용    | 두 번째 열의 내용    |
```

| 첫 번째 헤더      | 두 번째 헤더      |
| ----------------- | ----------------- |
| 첫 번째 셀의 내용 | 두 번째 셀의 내용 |
| 첫 번째 열의 내용 | 두 번째 열의 내용 |

<br />

## 취소선

두 개의 물결표( `~~` )로 감싼 단어는 ~~취소선~~이 적용됩니다.
