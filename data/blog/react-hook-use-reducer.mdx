# React의 useReducer 훅

React의 `useReducer` 훅은 상태 관리를 위해 사용되는 훅.  
이 훅은 전통적인 React 상태 관리 방법 중 하나인 `setState` 메서드 대신 사용될 수 있다.  
`useReducer`는 보다 복잡한 상태 로직을 다룰 때 유용하며, 특히 상태가 여러 번에 걸쳐 업데이트되거나 상태 간의 의존성이 있을 때 유용하다.

## 사용 예시

```jsx
import React, { useReducer } from 'react'

// 초기 상태를 정의
const initialState = {
  count: 0,
}

// reducer 함수를 정의
const reducer = (state, action) => {
  // action은 dispatch 함수를 통해 전달
  switch (action.type) {
    case 'increment':
      return { count: state.count + 1 }
    case 'decrement':
      return { count: state.count - 1 }
    default:
      return state
  }
}

const Counter = () => {
  // useReducer를 사용하여 상태와 dispatch 함수를 받음
  // dispatch 함수를 이용하여 컴포넌트에서 상태를 읽거나 변경 가능
  const [state, dispatch] = useReducer(reducer, initialState)

  return (
    <div>
      Count: {state.count}
      <button onClick={() => dispatch({ type: 'increment' })}>Increment</button>
      <button onClick={() => dispatch({ type: 'decrement' })}>Decrement</button>
    </div>
  )
}

export default Counter
```

## 정리

- useReducer는 상태를 업데이트 하기위해 reducer 함수를 사용한다.
- 상태를 어떻게 업데이트 할 지에 대한 내용을 action을 통해 전달한다.
- dispatch 함수를 호출하여 액션을 발생시키면, 해당 액션을 처리하는 reducer 함수가 호출되어 새로운 state를 반환한다.
- 컴포넌트의 상태 로직이 복잡해지거나 상태 간의 의존성이 있는 경우 useReducer를 사용하여 상태 관리를 할 수 있다.
